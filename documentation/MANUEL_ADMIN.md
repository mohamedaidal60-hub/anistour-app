# Manuel d'Utilisation - Profil ADMINISTRATEUR

## 1. Connexion et Rôles
L'administrateur possède les pleins pouvoirs sur l'application. Il est le seul habilité à voir les statistiques financières globales et à valider les opérations.

## 2. Le Tableau de Bord (Dashboard)
À votre connexion, vous voyez le "Live Profit" :
- **Recette Jour** : Ce qui est rentré aujourd'hui.
- **Dépenses Jour** : Les frais saisis.
- **Cash Global** : Le montant total théorique présent dans vos caisses agences.
- **Alertes KM** : Liste des véhicules qui doivent passer en entretien immédiatement.

## 3. Validation des Opérations
C'est votre tâche quotidienne la plus importante :
1. Allez dans l'onglet **"Validation"**.
2. Examinez les photos de preuves et les signatures laissées par les agents.
3. Si le montant est correct : Cliquez sur **"Approuver"**.
4. Si une erreur est faite : Cliquez sur **"Modifier & Valider"** pour corriger le montant ou la description.
5. Si l'opération est suspecte : Cliquez sur **"Rejeter"**.

## 4. Gestion de la Flotte (Menu Extra -> BI)
Cet onglet vous permet de prendre des décisions stratégiques :
- Consultez le graphique **"Intensité d'utilisation"** pour voir quels véhicules roulent le plus.
- Identifiez les véhicules les moins rentables (ceux qui coûtent trop cher en entretien par rapport à ce qu'ils rapportent).
- Gérez les documents (Assurances, etc.) en cliquant sur **"Gestion Documentaire"**.

## 5. Maintenance et Archivage (Sécurité)
Pour garantir la pérennité du système :
- **Backup mensuel** : Une fois par mois, cliquez sur **"Backup JSON"**. Le fichier téléchargé contient TOUTE votre entreprise. Gardez-le précieusement sur une clé USB.
- **Purge** : Si l'application affiche un message de saturation, faites un backup, puis cliquez sur **"Archivage & Purge"**. Le système repartira à zéro (rapide), mais vos anciennes données resteront dans votre fichier backup.

## 6. Gestion des Utilisateurs et Véhicules
Utilisez les boutons de configuration pour ajouter de nouveaux véhicules ou de nouveaux agents. N'oubliez pas de mettre une photo pour chaque véhicule pour une meilleure reconnaissance visuelle.

# Manuel d'Utilisation - Profil AGENT

## 1. Introduction
Bienvenue sur l'application Anistour. Votre rôle est de saisir quotidiennement les opérations de l'agence avec précision.

## 2. Saisie d'une Opération (Recette ou Dépense)
Pour chaque transaction (client qui paie, achat de carburant, petite réparation) :
1. Cliquez sur **"Nouvelle Entrée"**.
2. Choisissez le type : **RECETTE** (argent qui rentre) ou **DEPENSE** (argent qui sort).
3. Sélectionnez le **Véhicule** concerné (si c'est pour l'agence, laissez Vide/Général).
4. Saisissez le **Montant** exact.
5. Indiquez le **Kilométrage** actuel du véhicule (très important pour le suivi).
6. Prenez une **Photo du justificatif** (reçu, ticket de caisse).
7. Cliquez sur **"Signer ici"** : Signez sur l'écran avec votre doigt ou la souris.
8. Cliquez sur **"Enregistrer"**.

## 3. Le Registre (Journal)
Vous pouvez consulter vos saisies de la journée dans l'onglet **"Journal"**.
- Les opérations s'affichent avec le statut **"EN ATTENTE"** (Orange).
- Une fois que l'Admin a vérifié votre travail, le statut passe à **"VALIDE"** (Vert).
- Si le statut est **"REFUSE"** (Rouge), contactez l'Admin pour comprendre l'erreur.

## 4. Gestion des Documents
Dans le menu **"Extra"** -> **"GED"**, vous pouvez consulter les documents du véhicule (Assurance, Carte Grise) pour savoir s'ils sont encore valides avant de donner les clés à un chauffeur.

## 5. Règles d'or
- Ne jamais oublier de mettre l'index Kilométrique (KM).
- Toujours prendre une photo nette des reçus.
- Toujours faire signer la transaction sur le Pad Signature.

# Support de Formation : Anistour Fleet & Finance 2026

## [Diapositive 1 : Bienvenue]
- **Titre** : Anistour - Gestion de Flotte Intelligente
- **Slogan** : Efficacité, Transparence, Rentabilité.
- **Image suggérée** : Logo de l'entreprise avec une vue d'ensemble du Dashboard.

## [Diapositive 2 : Pourquoi cette application ?]
- **Problématique** : Trop de papiers, perte d'infos sur les entretiens, manque de vision sur le cash réel.
- **Solution** : Centralisation numérique, synchronisation en temps réel, alertes automatiques.

## [Diapositive 3 : Les 3 Piliers pour l'Agent]
1. **Saisir** tout de suite (ne pas attendre le soir).
2. **Preuve** visuelle (Photo du reçu).
3. **Engager** la responsabilité (Signature numérique).

## [Diapositive 4 : Démo Saisie (Animation)]
- Choix du véhicule -> Montant -> Photo -> Signature.
- *Message clé* : "Une saisie complète prend moins de 45 secondes."

## [Diapositive 5 : Les 3 Piliers pour l'Admin]
1. **Vérifier** : Ne plus faire confiance aveugle, regarder les preuves.
2. **Valider** : Débloquer les fonds dans la caisse d'un clic.
3. **Décider** : Utiliser les graphiques pour savoir quel véhicule vendre ou garder.

## [Diapositive 6 : Sécurité & Archivage]
- Le bouton **Backup JSON** est votre coffre-fort.
- L'archivage protège vos données contre les pertes ou les pannes internet.

## [Diapositive 7 : Conclusion & Questions]
- Une application moderne pour une entreprise leader.
- **Support technique** : Disponible via l'onglet MESSAGERIE intégré.
